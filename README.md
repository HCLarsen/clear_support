# Clear Support

General purpose extensions for the core Crystal library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     clear_support:
       github: HCLarsen/clear_support
   ```

2. Run `shards install`

## Usage

```crystal
require "clear_support"
```
